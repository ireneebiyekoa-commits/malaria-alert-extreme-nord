Guide d'utilisation

Mode d'emploi de la plateforme

Connexion

L'accès aux fonctionnalités de l'application nécessite une authentification. Deux rôles sont définis :

  • Administrateur : accès complet (KPI globaux, prévisions tous districts, import de données, génération de rapports Word).
  • Chef de district : accès restreint aux données de son district uniquement.

Tableau de bord

  1. Sélectionnez l'année et le district pour personnaliser l'affichage.
  2. La carte choroplèthe affiche l'incidence moyenne annuelle par district.
  3. Les graphiques de séries temporelles superposent l'incidence et les variables climatiques.
  4. La corrélation croisée (CCF) identifie les décalages temporels significatifs entre le climat et l'incidence.

Prévisions

  1. Sélectionnez le district, l'algorithme (RF, XGBoost, ou comparaison) et l'horizon de prévision.
  2. Cliquez sur « Générer les prévisions ».
  3. Consultez la courbe, le tableau des valeurs et les métriques du modèle.
  4. Cliquez sur « Générer l'analyse » pour obtenir une interprétation IA.
  5. Les administrateurs peuvent exporter un rapport Word complet.

Carte d'alerte

  1. Choisissez l'algorithme et l'horizon de prévision.
  2. Cliquez sur « Actualiser » pour calculer les niveaux d'alerte.
  3. La carte affiche les districts colorés selon leur niveau (vert / orange / rouge).
  4. Le panneau latéral liste les districts triés par sévérité décroissante.
  5. Cliquez sur « Export PDF » pour intégrer la carte dans les rapports du GTR.

Mise à jour des données (Admin)

  1. Téléchargez le template Excel du mois à saisir.
  2. Renseignez la colonne cas_confirmes pour chaque district.
  3. Importez le fichier complété : l'application récupère automatiquement le climat via NASA POWER et recalcule les prévisions.
  4. Le journal des imports trace toutes les opérations.
