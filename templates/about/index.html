À propos du système

Présentation, objectifs et contexte institutionnel

Mission

Le {{ APP_NAME }} est une plateforme institutionnelle d'aide à la décision conçue par l'{{ APP_AUTHOR }}, au sein de la {{ APP_DELEGATION }}, sous tutelle du {{ APP_INSTITUTION }}.

Elle vise à anticiper la dynamique épidémique du paludisme dans les 32 districts sanitaires de la région à l'aide de modèles d'apprentissage automatique (Random Forest et XGBoost), entraînés sur huit années de données épidémiologiques et climatiques (2017-2025).

Composantes fonctionnelles

  • Tableau de bord : indicateurs clés, cartographie d'incidence, séries temporelles, corrélations climat-incidence.
  • Prévisions : projections à 1, 2 et 3 mois par district, avec analyse IA et génération de rapports Word.
  • Carte d'alerte : niveaux vert / orange / rouge basés sur les seuils d'alerte (M + σ) et épidémiologique (M + 2σ).
  • Mise à jour des données : import mensuel + récupération automatique du climat via NASA POWER.

Stack technique

BackendPython · Django 4.2 (patron MVT)
Base de donnéesSQLite (dev) / PostgreSQL (production)
Machine Learningscikit-learn (Random Forest), XGBoost, joblib
VisualisationChart.js, Leaflet.js
IA générativeGoogle Gemini API
Données climatiquesNASA POWER API
